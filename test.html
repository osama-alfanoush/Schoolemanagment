1. Culture Media Classification

Media are formulated based on physical consistency and functional parameters to support distinct microbial nutritional requirements:

Media Type Agar % Practical Use
Liquid (Broth) 0% Rapid biomass increase (e.g., Nutrient Broth)
Semisolid 1% Flagellar motility testing (MIO Agar)
Solid 1.5% - 2% Isolate pure clones & study morphology

Functional Categories:

  • Simple Medium: Supports non-fastidious organisms (Nutrient Agar).
  • Enriched Medium: Added whole blood/serum for fastidious pathogens (Blood Agar).
  • Differential Medium: Distinguishes species via metabolic traits (MacConkey differentiates lactose status).
  • Selective Medium: Inhibits specific groups while allowing targets to grow (SS Agar for Salmonella/Shigella).

2. Gram Stain Workflow

The definitive primary classification test based on bacterial cell wall integrity and biochemistry:

  1. Primary Stain: Crystal Violet (1 min) - Stains all cells blue/purple.
  2. Mordant: Iodine (1 min) - Forms CV-I complex inside cells.
  3. Decolorizer: Alcohol (15 sec) - Extracts outer lipids in Gram-negative walls.
  4. Counterstain: Safranin (1 min) - Stains decolorized structures.
Reaction Microscopic Color Structural Rationale
Gram-Positive (G+) Purple / Blue Thick peptidoglycan layer retains CV-I complex.
Gram-Negative (G-) Pink / Red Thin peptidoglycan + lipid-rich outer membrane washed by alcohol.

3. Gram-Positive Cocci Identification

The identification tree begins by distinguishing catalase production across spherical bacteria.

Catalase Test: Application of Hydrogen Peroxide (H2O2) onto fresh isolated colonies:

  • Positive (+) Vigorous effervescence / oxygen gas bubbles → Staphylococcus (clusters).
  • Negative (-) Absence of bubbling → Streptococcus (chains).
